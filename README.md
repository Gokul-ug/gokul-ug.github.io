# Personal Portfolio — GOKUL U

> A clean, responsive personal portfolio showcasing software, IoT and embedded projects.

## Overview

This repository contains a static personal portfolio website built with HTML, CSS, and a small JavaScript file for interactions. It highlights projects, skills, experience, and contact information for GOKUL U.

## Features

- Responsive, modern layout with nav, sections, and project cards.
- Project flip cards in the portfolio section.
- Theme toggle and back-to-top button (controlled by `main.js`).

## Demo / Preview

Open `index.html` in a browser to view the site locally.

## Local setup

1. Clone or download this repository.
2. Open the project folder in your file explorer or editor.
3. Open `index.html` in your browser (double-click or use `Live Server`).

Quick command (if you have a simple HTTP server installed):

```
# serve from project root (example using Python 3)
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

## Project structure

- `index.html` — main site HTML
- `styles.css` — global styles and layout
- `main.js` — small scripts for interactions and theme toggling
- `assets/` — images used across the site (header, project thumbnails, etc.)

## Customize

- Replace images in `assets/` with your own (keep the same filenames or update paths in `index.html`).
- Update text content (name, bio, links) directly in `index.html`.
- Modify colors and spacing in `styles.css`.
